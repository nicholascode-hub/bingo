# 🎱 Jogo de Bingo em Python

Um jogo de Bingo interativo e com interface gráfica moderna, desenvolvido totalmente em Python. Este projeto permite partidas de 1 a 4 jogadores com layouts e cartelas geradas dinamicamente!

## ✨ Funcionalidades

* **Múltiplos Jogadores:** Suporte para 1 a 4 jogadores simultâneos na mesma tela.
* **Interface Dinâmica:** O tamanho da janela e a disposição das cartelas (grid) se adaptam automaticamente à quantidade de jogadores escolhida.
* **Sorteio Automático:** Controle interno dos números disponíveis (1 a 75) para evitar repetições durante o sorteio.
* **Design Moderno:** Interface gráfica aprimorada utilizando a biblioteca `ttkbootstrap`, saindo do visual padrão do Tkinter clássico.

## 🛠️ Tecnologias Utilizadas

* **Python 3.x:** Linguagem principal do projeto.
* **Tkinter:** Biblioteca padrão do Python para criação da interface gráfica (GUI).
* **ttkbootstrap:** Extensão temática para o Tkinter, trazendo componentes modernos.
* **Random:** Biblioteca nativa para a lógica de sorteio aleatório dos números.
